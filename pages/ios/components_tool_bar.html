Tool Bar

A Tool Bar contains actionable controls related to the items on the screen or view

It can be useful to display a filter/sort bar when there are clearly defined or typical categories in which users might want to refine their search. The filter/sort bar is displayed anchored to the bottom of the page and displays the number of results as well as the filter/sort buttons

Specs

Font, color, and Usage:

  • Background fill: White #FFFFFF
  • Title: Helvetica Regular, 16 points
  • "Delete" text color: Red #BF6767
  • "Add Reports, Filter, Sort" text color: Concur Blue #0078C8
  • "30 results" text color: Medium Grey #A7B6BF
  • Tool Bar: 44 tall
  • Text: 15 padding
  • Text Alignment: centerY
