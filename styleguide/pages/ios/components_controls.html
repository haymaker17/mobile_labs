Date Picker

The date picker uses the familiar calendar pattern. If a single date is required, a user may tap on the date to fill in the date form. If a date range is required, a user may tap on the start date and then the end date, or tap the start date and drag to the end date.

Multi-Select (thumbnails)

When multiple selections are requred in a thumbnail view, the user first taps the "select" icon and the view is put in "select mode". Now the user may select one or multiple items by tapping on them and then use the contextual actions that appear on the bottom of the screen to act on those items.

Multi-Select (list)

When multiple selections are requred in a list view, the user first taps the "select" icon and the view is put in "select mode". Now the user may select one or multiple items by tapping on them and then use the contextual actions that appear on the bottom of the screen to act on those items.
